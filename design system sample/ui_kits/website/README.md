# Northvale Capital · Website UI Kit

A high-fidelity recreation of the Northvale Capital marketing-website pattern: editorial top nav, restrained hero with serif display headline, insights grid, market data callout, and a quiet footer.

## Components

- `Header.jsx` — top utility bar + primary nav with the wordmark
- `Hero.jsx` — full-width editorial hero (headline + supporting copy + CTA)
- `InsightsGrid.jsx` — 3-up card grid of editorial articles
- `MarketStrip.jsx` — live-feel ticker strip with tabular numerals
- `CalloutBand.jsx` — dark inverse band for a featured story
- `Footer.jsx` — multi-column footer with the wordmark and disclosures
